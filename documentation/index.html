Installation


  • Recommended:
    Make sure you have Node.js installed on your computer. We recommend version 18.17 or later. If you don't have it yet, you can download and install it from the official Node.js website
  • Use command line (powershell or terminal) to navigate to project root directory.
  • Install dependencies:
    Run the following command to download and install the necessary packages for your project:
    npm install
  • Run the project in development mode:
    Once the installation is complete, use the following command to start your project in development mode and Open http://localhost:3000 with your browser to see the result
    npm run dev

Favicon & Logo


  • Favicon
    Replace the favicon.ico with the .ico file that you created for your website. Favicon.ico is located in the ./src/app folder, located at the root folder.
  • logo
    To change the logo, follow the directory "./public/image/logo/" and replace the logo.png with your amazing brand logo.
